Since the intermediate output has been saved and is provided, one can generate Figures A1 and A2 in the paper by running figureA1.m and figureA2.m.
The corresponding results in Table 1 and 2 are generated by running table_hd and table_vdc.

The code is set up to run simulations for different seeds, saving each result, and combining these results. 
To generate the simulation results, run main.m, making sure to change the output file name and the seed, as needed. 
Then combine the results using combine.m. Then create the figures and tables, following the instructions above.
